# Emoji Font Sets for GraveGain 2D

Place emoji font `.ttf` files in this folder to enable alternative emoji styles.

## Supported Emoji Sets

| Set | Font File | License | Download |
|-----|-----------|---------|----------|
| **Twemoji** | `Twemoji.Mozilla.ttf` | CC BY 4.0 / Apache 2.0 | [mozilla/twemoji-colr](https://github.com/nicedoc/twemoji-colr/releases) |
| **Noto Color Emoji** | `NotoColorEmoji.ttf` | Apache 2.0 | [googlefonts/noto-emoji](https://github.com/googlefonts/noto-emoji/releases) |
| **OpenMoji** | `OpenMoji-Color.ttf` | CC BY-SA 4.0 | [hfg-gmuend/openmoji](https://github.com/hfg-gmuend/openmoji/releases) |
| **Blobmoji** | `Blobmoji.ttf` | Apache 2.0 | [C1710/blobmoji](https://github.com/C1710/blobmoji/releases) |
| **Fluent Emoji** | `FluentSystemEmoji.ttf` | MIT | [AcmeSoftwareLLC/fluent-emoji-font](https://github.com/AcmeSoftwareLLC/fluent-emoji-font/releases) |
| **JoyPixels** | `joypixels-android.ttf` | Free License | [joypixels.com](https://www.joypixels.com/download) |
| **Samsung Emoji** | `SamsungColorEmoji.ttf` | Proprietary | Extract from device |

## How to Install

1. Download the `.ttf` file from the link above
2. Place it in this folder (`fonts/emoji/`)
3. Launch the game and go to Graphics Settings
4. Click "Rescan for Fonts" if needed
5. Select your preferred emoji set

## Alternative Location

You can also place fonts in `user://fonts/emoji/` which is:
- **Windows**: `%APPDATA%/Godot/app_userdata/GraveGain 2D/fonts/emoji/`
- **macOS**: `~/Library/Application Support/Godot/app_userdata/GraveGain 2D/fonts/emoji/`
- **Linux**: `~/.local/share/godot/app_userdata/GraveGain 2D/fonts/emoji/`
